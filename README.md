# Writing Verilog Codes with icarus verilog 

### Steps to edit and run the code
Step-1: Write a verilog code

```
$ gedit hello1.v
```

Step-2: Compiling a verilog code with icarus verilog

```
$ iverilog hello1.v
```

Step-3: run the code

```
$ vvp a.out
```
